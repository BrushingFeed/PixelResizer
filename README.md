# PixelResizer
無劣化でピクセル画像を拡大するソフト

## ライセンス
MIT ライセンス
著作権（C）01349net, cr47t

## システム要件
* .NET 8.0が導入されたx86またはx64システム
* 最低200KBのディスクスペース
* 最低500MBのメモリ

## 使用方法
* **画像を読み込む**をクリックして処理する画像を選択する
* **倍率**を1から100までの範囲で変更する
* **画像を保存する**をクリックして処理後の画像を保存する
